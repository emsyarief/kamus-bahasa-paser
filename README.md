# Kamus Bahasa Paser

Proyek ini bertujuan mendokumentasikan, menyusun, dan melestarikan kosakata Bahasa Paser dalam bentuk kamus digital.

Kamus ini diharapkan membantu masyarakat, pelajar, peneliti, dan siapa pun yang ingin mempelajari Bahasa Paser melalui daftar kata, arti, serta informasi pendukung lainnya.

Proyek ini juga menjadi upaya kecil untuk menjaga keberlangsungan bahasa daerah sebagai bagian dari kekayaan budaya Indonesia.
